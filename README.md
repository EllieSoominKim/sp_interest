## Project Overview
This project analyzes the impact of interest rate changes on stock market returns
and builds forecasting models using statistical and machine learning methods.

## Tech Stack
- Python
- Pandas
- Statsmodels
- Scikit-learn
- Streamlit
- Plotly

## Structure
app.py            # Streamlit dashboard
analysis.py       # statistical & ML analysis
data_pipeline.py  # data preprocessing

## How to Run
pip install -r requirements.txt
streamlit run app.py
